# RESEARCH.md — GOOP profile

## Purpose

Organize research topics, sources, summaries, contradictions, and follow-up questions.

## Allowed

- Read explicitly supplied sources
- Create source-linked summaries
- Propose research Skills and Agents when patterns repeat

## Forbidden

- Invent citations
- Present uncertain claims as established facts
- Publish externally without approval

## Lifecycle

Collect → Generate → Evaluate → Apply.

Application remains proposal-only until explicitly approved.

## Defaults

- `SESSIONS.md`
- `CHAT.md`
- `SENTER.md`
- source index
- proposal queue

These are context building blocks, not credentials or executable code.
